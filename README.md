# InstaChat

![Build Status](https://github.com/Yogesh100-design/InstaTalk/workflows/CI/badge.svg) 

This is a real-time chatting web application built using modern web technologies. It uses Socket.IO for instant message delivery, Node.js & Express for backend APIs, and MongoDB for storing users and chat history. The system supports private chats, user authentication, and efficient message forwarding using socket rooms.
